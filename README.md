🚗 SGX Mobility Mobile App

SGX Mobility is a smart transportation platform designed to enhance mobility services by connecting users with efficient, reliable, and data-driven transport solutions.

📱 Features

🔐 User Authentication (Sign up / Login)

📍 Real-time Location Tracking

🚕 Ride Booking System

💳 Payment Integration

📊 Ride History & Analytics

🔔 Push Notifications

🤖 AI-based Recommendations (optional/future)

🛠️ Tech Stack

Frontend: Flutter

Backend: Firebase (Authentication, Firestore, Realtime DB)

State Management: Provider / Riverpod (choose what you used)

Maps & Location: Google Maps API

Notifications: Firebase Cloud Messaging

🚀 Getting Started
Prerequisites

Flutter SDK installed

Dart installed

Android Studio / VS Code

Firebase project setup

🔧 Installation

Clone the repository

git clone https://github.com/your-username/sgx-mobility.git
cd sgx-mobilit
Install dependencies

flutter pub get

Run the app

flutter run
🔥 Firebase Setup

Create a project in Firebase Console

Add Android/iOS apps

Download google-services.json / GoogleService-Info.plist

Enable:
Authentication
Firestore Database
Cloud Messaging
📍 Environment Configuration
Create a .env file:
GOOGLE_MAPS_API_KEY=your_api_key
FIREBASE_API_KEY=your_key
🧪 Testing
flutter test
